# 总结
* `meta name='viewport'`：抄淘宝
* `iconfont`使用
* 用负 `margin` 实现平均布局：子容器 `margin-right:10px` 父容器 `margin-right:-10px`
* 媒体查询响应式布局
```CSS
@media (min-width:500px){

}
```
* `localStorage`
```javaScript
localStorage.getItem(key)
localStorage.setItem(key, value)
```
* `JSON`
```javaScript
JSON.stringify(arr) //数组转化为字符串
JSON.parse(str) //字符串转化为数组
```
* 正则表达式：https://deerchao.cn/tutorials/regex/regex.htm
